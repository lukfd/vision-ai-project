# Predict Restaurant Ratings

## Dataset

The dataset is from the website yelp.com which publishes user sourced reviews about businesses around the world. The dataset is a subset of the Yelp data, and contains real-world information about businesses such as user reviews, users' photos, restaurants descriptions...

<https://business.yelp.com/data/resources/open-dataset/>

Dataset description: The Yelp dataset includes over 6 million reviews from customers and 200,100 images from over 150,000 businesses. There are 4 attributes:

- photo_id, business_id, caption, and label
- business_id, Name, address, city, state, postal_code, latitude, longitude, stars, review_count, is_open, attributes, categories, hours, Review_id
